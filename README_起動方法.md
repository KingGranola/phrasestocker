# PhraseStocker 起動方法

## 🚀 ワンクリックで起動

### 方法1: ダブルクリックで起動（推奨）

1. Finderでプロジェクトフォルダを開く
2. **`start-dev-server.command`** ファイルをダブルクリック
3. ブラウザが自動的に開きます（`http://localhost:5173`）

> 💡 初回起動時のみ、依存関係のインストールに数分かかります。

### 停止方法

- ターミナルウィンドウを閉じる
- または `Ctrl + C` を押す

---

## 📝 注意事項

### 初回起動時

「開発元が未確認のため開けません」というメッセージが表示された場合:

1. **システム設定** → **プライバシーとセキュリティ** を開く
2. 「このまま開く」ボタンをクリック
3. 確認ダイアログで「開く」をクリック

### ポートが使用中の場合

別のアプリが5173番ポートを使用している場合:

1. 起動スクリプトを停止
2. 他のアプリを終了
3. 再度起動スクリプトを実行

---

## 🔧 トラブルシューティング

### ブラウザが開かない場合

手動で以下のURLにアクセス:
```
http://localhost:5173
```

### エラーが表示される場合

1. ターミナルウィンドウのエラーメッセージをコピー
2. 開発者に共有
